# slop-review for OpenCode

OpenCode port of the `slop-review` plugin.

## Included Runtime Files

- `skills/ai-slop-review/SKILL.md`
- `skills/ai-slop-review/references/`
- `commands/slop-review-review.md`

## Install

Use the repository install entrypoint:

```text
Fetch and follow instructions from https://raw.githubusercontent.com/sentiolabs/agent-nexus/main/.opencode/INSTALL.md
```

Then choose `slop-review` or `all`.

## OpenCode Surface

- Skill: `ai-slop-review`
- Command: `/slop-review-review`

See `docs/marketplaces/platform-support-matrix.md` for the current support model and install
constraints.
